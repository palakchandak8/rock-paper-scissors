# Desktop OpenCV Version

This folder contains the standalone desktop version of Rock Paper Scissors that uses OpenCV GUI windows instead of a web interface.

## üñ•Ô∏è OpenCV Desktop Version

**File:** `rps-opencv.py`

### Features:
- Uses OpenCV window for display
- Real-time hand gesture detection
- Countdown timer (3-2-1)
- Press SPACE to start a round
- Press ESC to exit

### Requirements:
```bash
pip install opencv-python mediapipe
```

### How to Run:
```bash
cd desktops
python rps-opencv.py
```

### Controls:
- **SPACE**: Start a new round
- **ESC**: Exit the game

---

**Note:** This is the older desktop version. For the modern web-based version with better UI, use the main application in the `frontend/` and `backend/` folders.
